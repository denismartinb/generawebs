# Assets del generador

- `layouts/`: miniaturas y recursos visuales de cada layout predefinido.
- `sections/`: placeholders y recursos por tipo de sección.

Estructura recomendada para ampliar:

- `assets/layouts/<layout-id>/thumbnail.svg`
- `assets/layouts/<layout-id>/preview.svg`
- `assets/layouts/<layout-id>/...`
- `assets/sections/<section-id>/placeholder.svg`
- `assets/sections/<section-id>/...`
